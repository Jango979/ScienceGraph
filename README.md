# ScienceGraph

Local tool for organizing and exporting scientific figures for research articles.

## Features

- Load images one by one (PNG, TIFF, JPEG, BMP, SVG, EPS, PDF)
- Edit: rotate, flip, adjust brightness and contrast
- Export in multiple formats and compression settings:
  - TIFF (uncompressed / LZW lossless)
  - PNG (uncompressed / compressed)
  - JPEG (high quality / publication)
  - PDF, EPS

## Setup

```bash
pip install -r requirements.txt
python -m src.main
```

## Requirements

- Python 3.10+
- See `requirements.txt`
